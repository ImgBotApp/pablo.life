# pablo.life

_Minimalist personal website. Inpsired (a lot) by https://github.com/rauchg/blog_

```shell
cd pablo.life/
yarn install
npm run build
npm run start
```

# How

It's made with [next.js](https://github.com/zeit/next.js).

```Shell
npm export # creates static folder `out`
```
